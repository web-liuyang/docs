# 用户管理命令
## useradd

命令名称：**useradd**

命令英文原意：**useradd**

命令所在路径：**/usr/sbin/useradd**

功能描述：添加新用户

语法：**useradd [用户名]**

```shell
useradd liuyang
```

## passwd

命令名称：**passwd**

命令英文原意：**password**

命令所在路径：**/usr/bin/passwd**

功能描述：修改用户密码

语法：**passwd [用户名]**

```shell
passwd liuyang
```
